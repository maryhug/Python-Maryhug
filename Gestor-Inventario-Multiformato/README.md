## Gestor de Inventario Multiformato
Proyecto modular en Python para gestionar un inventario básico, cumpliendo:
- Registrar productos
- Listarlos
- Editarlos
- Eliminarlos
- Guardar y cargar desde **JSON, CSV y TXT**
- Estructura totalmente modular
- Funciones de máximo 15 líneas

## Formatos soportados
- JSON
- CSV
- TXT
